This is a finite element program of planery triangular element, it takes much work.
I write a general 2d triangle solver, and write a simple shape with 21 * 11 grid,
and output those displacements.

The program is written in Linux system, complied by gcc with standard C++ 17.

Dependencies:
Eigen: see https://gitlab.com/libeigen/eigen
